# 5. Modificar nodos en el DOM
Una vez que hemos encontrado los nodos, podemos modificarlos. Algunos métodos útiles incluyen:

- `.innerHTML`, `.innerText`, `.outerHTML`: Permite modificar el contenido HTML o texto de un elemento.
- `.insertAdjacentHTML(position, text)`: Permite inserta texto HTML en una posición específica.
- `.append(content, element)`, `.prepend(content, element)`: Permite añadir contenido al principio o al final de un elemento.
- `.after()`, `.before()`: Permite insertar un elemento antes o después del elemento actual.
- `.cloneNode(deep)`: Permite clonar un nodo, con o sin sus hijos.
- `.remove()`: Permite eliminar un nodo.

Métodos más antiguos pero aún en uso incluyen `removeChild()` y `appendChild()`.

Más elemento en: [Elementos del DOM](https://www.w3schools.com/jsref/dom_obj_all.asp)

### 5.1.2 insertAdjacentHTML

Método que permite insertar código HTML en una posición específica. Las posición donde se puede insertar código son:

- `afterbegin`: Después del inicio del elemento (primer hijo).
- `afterend`: Después del elemento.
- `beforebegin`: Antes del elemento.
- `beforeend`: Antes del final del elemento (último hijo).

!!! example "Ejemplo 1: Insertar un nuevo elemento `<p>` después de un elemento de encabezado"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>
            <h2>Método insertAdjacentHTML</h2>

            <button onclick="myFunction()">Insertar</button>
            <p>Haz clic en "Insertar" para insertar un párrafo después del encabezado.</p>

            <h2 id="myH2">Mi encabezado</h2>

            <script>
                function myFunction() {
                    const h2 = document.getElementById("myH2");
                    let html = "<p>My new paragraph.</p>";
                    h2.insertAdjacentHTML("afterend", html);
                }
            </script>
        </body>
    </html>
    ```

### 5.1.3 append, prepend

#### 4.1.3.1 append

El método `append` se utiliza para agregar uno o varios nodos (elementos) o cadenas después del último hijo de un elemento.

!!! example "Ejemplo 1: Agregar texto a un div:"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>
            <h2>Método append</h2>

            <div id="myDiv"></div>

            <p>Haz clic en "Añadir" para agregar texto a un div:</p>

            <button onclick="myFunction()">Añadir</button>

            <script>
                function myFunction() {
                    const div = document.getElementById("myDiv");
                    div.append("Hola! Soy un texto adjunto.");
                }
            </script>
        </body>
    </html>
    ```

!!! example "Ejemplo 2: Agregar un elemento a un div:"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>
            <h2>Método append</h2>

            <p id="myP">Segundo elemento</p>

            <div id="myDiv"><h3>Primer elemento</h3></div>

            <p>Haz clic en "Añadir" para agregar elementoa un div:</p>

            <button onclick="myFunction()">Añadir</button>

            <script>
                function myFunction() {
                    const div = document.getElementById("myDiv");
                    const p = document.getElementById("myP");
                    div.append(p);
                }
            </script>
        </body>
    </html>
    ```

### 5.1.5 cloneNode

El método `cloneNode(deep)` crea una copia de un nodo y devuelve el clon.

!!! example "Ejemplo 1: Copia un elemento `<li>` desde myList2 a myList1"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>
            <h2>Método cloneNode</h2>

            <button onclick="myFunction()">Copiar</button>
            <p>Haz clic en "Copiar" para copiar un elemento de una lista a otra.</p>

            <ul id="myList1">
                <li>Coffee</li>
                <li>Tea</li>
            </ul>
            <ul id="myList2">
                <li>Water</li>
                <li>Leche</li>
            </ul>

            <p>Si cambia el parámetro <b>deep</b> a false, solo se clonará un elemento LI vacío.</p>

            <script>
                function myFunction() {
                    const node = document.getElementById("myList2").lastChild;
                    const clone = node.cloneNode(true);
                    document.getElementById("myList1").appendChild(clone);
                }
            </script>
        </body>
    </html>
    ```

## 5.2 Atributos

Los elementos suelen tener **atributos**. Algunos son especiales como `id` o `class`. El **id** está accesible directamente como atributo del elemento, así como el **className**, aunque luego veremos que es mejor manipularlo de otra manera. Otros atributos como **value** en los `Input` o **src** en los `<img>` también pueden ser leídos y modificados de forma directa.

Sin embargo, para los atributos que no tienen acceso directo porque no son estándar, podemos usar `setAttribute()`, `getAttribute()`, `hasAttribute()` o `removeAttribute()`:

!!! example "Ejemplo 1: Acceso a los atributos de los elementos"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>

            <button>Botón</button>

            <script>
                const button = document.querySelector("button");
                console.log("Valor del atributo 'nombre' del elemento 'botón': " + button.getAttribute("name"));
                button.setAttribute("name", "helloButton");
                button.setAttribute("disabled", "");
            </script>
        </body>
    </html>
    ```

## 5.3 Estilos

Para manipular los estilos de un elemento, podemos usar propiedades de estilo y clases CSS:

- `.style.property`: Modifica un estilo CSS directamente.
- `.className`: Cambia el nombre de la clase del elemento.
- `.classList.add()`, `.classList.toggle()`, `.classList.remove()`, `.classList.replace()`: Métodos para manipular clases CSS de manera más dinámica.

!!! example "Ejemplo 1: Manipulación del CSS"
    ```html
    <!DOCTYPE html>
    <html>
        <body>
            <h1>Element Object Model</h1>

            <div id="myId">Manipulación de estilos CSS</div>

            <script>
                let element = document.getElementById('exampleId');
                element.style.color = 'blue';
                element.classList.add('new-class');
                element.classList.remove('old-class');
            </script>
        </body>
    </html>
    ```

            function eliminar() {
                let i = 0;
                let texto, precio, items

                // 1. obtener los elementos hijos del elemento <ul id="lista">, los cuales son de tipo <li> 
                items = document.querySelector("#lista").children;

            function añadir() {

                let texto = document.querySelector("#tarea").value;

                let li = document.createElement("li");
                li.textContent = texto;

                // Clase importante
                if (document.querySelector("#importante").checked) {
                    li.classList.add("importante");
                }

                // Botón eliminar
                let btn = document.createElement("button");
                btn.textContent = "X";

                btn.onclick = function(e) {
                    e.stopPropagation();
                    li.remove();
                };

                li.append(btn);

                document.querySelector("#lista").append(li);

                document.querySelector("#tarea").value = "";
            }
