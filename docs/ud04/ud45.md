# 5. Modificar nodos en el DOM
Una vez que hemos encontrado los nodos, podemos modificarlos. Algunos métodos útiles incluyen:

- `.innerHTML`, `.innerText`, `.outerHTML`: Permite modificar el contenido HTML o texto de un elemento.
- `.insertAdjacentHTML(position, text)`: Permite inserta texto HTML en una posición específica.
- `.append(content, element)`, `.prepend(content, element)`: Permite añadir contenido al principio o al final de un elemento.
- `.after()`, `.before()`: Permite insertar un elemento antes o después del elemento actual.
- `.cloneNode(deep)`: Permite clonar un nodo, con o sin sus hijos.
- `.remove()`: Permite eliminar un nodo.

Métodos más antiguos pero aún en uso incluyen `removeChild()` y `appendChild()`.

Más elemento en: [Elementos del DOM](https://www.w3schools.com/jsref/dom_obj_all.asp)

### 5.1.2 insertAdjacentHTML

Método que permite insertar código HTML en una posición específica. Las posición donde se puede insertar código son:

- `afterbegin`: Después del inicio del elemento (primer hijo).
- `afterend`: Después del elemento.
- `beforebegin`: Antes del elemento.
- `beforeend`: Antes del final del elemento (último hijo).

??? example "Ejemplo 1 - Insertar un nuevo elemento `<p>` después de un elemento"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <script src="app.js" defer></script>
            </head>
            <body>
                <h1>Element Object Model</h1>
                <h2>Método insertAdjacentHTML</h2>

                <button onclick="myFunction()">Insertar</button>
                <p>Haz clic en "Insertar" para insertar un párrafo después del encabezado.</p>

                <h2 id="myH2">Mi encabezado</h2>
            </body>
        </html>
        ```
    === "app.js"

        ```javascript
        function myFunction() {
            let h2 = document.getElementById("myH2");
            let html = "<p>My new paragraph.</p>";

            h2.insertAdjacentHTML("afterend", html);
        }
        ```

### 5.1.3 append, prepend

#### 4.1.3.1 append

El método `append` se utiliza para agregar uno o varios nodos (elementos) o cadenas después del último hijo de un elemento.

??? example "Ejemplo 1 - Agregar un elemento después de un div"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <script src="app.js" defer></script>
            </head>
            <body>
                <h1>Element Object Model</h1>
                <h2>Método append</h2>

                <div id="myDiv"><h3>Primer elemento</h3></div>

                <p>Haz clic en "Añadir" para agregar elementoa un div:</p>

                <button onclick="myFunction()">Añadir</button>
            </body>
        </html>
        ```

    === "app.js"

        ```javascript
        function myFunction() {

            div.append(nuevoParrafo);
        }
        ```

### 5.1.5 cloneNode

El método `cloneNode(deep)` crea una copia de un nodo y devuelve el clon.

??? example "Ejemplo 1 - Copia un elemento `<li>` desde myList2 a myList1"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <script src="app.js" defer></script>
            </head>
            <body>
                <h1>Element Object Model</h1>
                <h2>Método cloneNode</h2>

                <button onclick="myFunction()">Copiar</button>
                <p>Haz clic en "Copiar" para copiar un elemento de una lista a otra.</p>

                <ul id="myList1">
                    <li>Coffee</li>
                    <li>Tea</li>
                </ul>
                <ul id="myList2">
                    <li>Water</li>
                    <li>Leche</li>
                </ul>

                <p>Si cambia el parámetro <b>deep</b> a false, solo se clonará un elemento LI vacío.</p>

        ```
    === "app.js"

        ```javascript
            let node = document.querySelector("#myList2").lastElementChild;
            let clone = node.cloneNode(true);

            document.querySelector("#myList1").appendChild(clone);
        }
        ```

## 5.2 Atributos

Los elementos suelen tener **atributos**. Algunos son especiales como `id` o `class`. El **id** está accesible directamente como atributo del elemento, así como el **className**, aunque luego veremos que es mejor manipularlo de otra manera. Otros atributos como **value** en los `Input` o **src** en los `<img>` también pueden ser leídos y modificados de forma directa.

Sin embargo, para los atributos que no tienen acceso directo porque no son estándar, podemos usar `setAttribute()`, `getAttribute()`, `hasAttribute()` o `removeAttribute()`:

??? example "Ejemplo 1 - Acceso a los atributos de los elementos"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <script src="app.js" defer></script>
            </head>
            <body>
                <h1>Element Object Model</h1>

                <button onclick="myFunction()">Botón</button>
                <p id="myP"></p>
            </body>
        </html>
        ```
    === "app.js"

        ```javascript
        function myFunction() {
            let button = document.querySelector("button");
            
            button.setAttribute("name", "helloButton");
            button.disabled = true;

            document.querySelector("#myP").textContent = button.getAttribute("name");
        }
        ```

## 5.3 Estilos

Para manipular los estilos de un elemento, podemos usar propiedades de estilo y clases CSS:

- `.style.property`: Modifica un estilo CSS directamente.
- `.className`: Cambia el nombre de la clase del elemento.
- `.classList.add()`, `.classList.toggle()`, `.classList.remove()`, `.classList.replace()`: Métodos para manipular clases CSS de manera más dinámica.

??? example "Ejemplo 1 - Acceso a los atributos de los elementos"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <html lang="es">
            <head>
                <meta charset="UTF-8">
                <script src="app.js" defer></script>
                <link rel="stylesheet" href="estilos.css">
            </head>
            <body>
                <h1>Element Object Model</h1>

                <div id="myId">Manipulación de estilos CSS</div>

                <button onclick="anyadirColor()">Añadir Color</button>
                <button onclick="asignarCSS()">Asignar CSS</button>
                <button onclick="borrarCSS()">Eliminar CSS</button>
                <button onclick="anyadirCSS()">Añadir CSS</button>
            </body>
        </html>
        ```
    === "app.js"

        ```javascript
        function anyadirColor() {
            let element = document.querySelector('#myId');

            element.style.color = 'red';
        }

        function asignarCSS() {
            let element = document.querySelector('#myId');

            element.style.color = '';
            element.className = 'normal';
        }

        function borrarCSS() {
            let element = document.querySelector('#myId');

            element.style.color = '';
            element.classList.remove('normal');
        }

        function anyadirCSS() {
            let element = document.querySelector('#myId');

            element.style.color = '';
            element.classList.add('tamanyo');
        }
        ```

    === "estilos.css"

        ```css
        .normal {
            text-decoration: line-through;
            color: gray;
        }

        .importante {
            color: blue;
            font-weight: bold;
        }

        .tamanyo {
            font-size: 20px;
        }
        ```

### 5.4.2 append y prepend

!!! note "Ejercicios con `append` y `prepend`"

    ??? example "Ejercicio 1: Crear una lista de la compra donde se añadan productos a una lista."
                
        === "index.html"

            ```html
            <!DOCTYPE html>
            <html lang="es">
            ```

            ```javascript
            function anyadir() {

                // 2. Obtener el elemento <ul id="lista"></ul> y almacenarlo en la variable 'lista'.

                // 3. Crear un elemento de tipo <li> y asignarlo a la variable 'li'

                // 4. Asignar al elemento 'li' creado en el punto 3 el texto obtenido en el punto 1.

                // 5. Añadir el elemento creado en el punto 3 al elemento obtenido en el punto 2. Emplear 'append'
            ```

            ```javascript
            function anyadir() {
                let texto, lista, li;

                // 1. Obtener el valor del input que tenga como id = producto. Almacenar el texto recuperado en la varibale texto.
                texto = document.querySelector("#producto").value;

                // 2. Obtener el elemento <ul id="lista"></ul> y almacenarlo en la variable 'lista'.
                lista = document.querySelector("#lista");

                // 3. Crear un elemento de tipo <li> y asignarlo a la variable 'li'
                li = document.createElement("li");

                // 4. Asignar al elemento 'li' creado en el punto 3 el texto obtenido en el punto 1.
                li.textContent = texto;

                // 5. Añadir el elemento creado en el punto 3 al elemento obtenido en el punto 2. Emplear 'append'
                lista.append(li);
            }
            ```

    ??? example "Ejercicio 2: Crear una lista con tareas urgentes"
                
        === "index.html"

            ```html
            <!DOCTYPE html>
            <html lang="es">
                <head>
                    <meta charset="UTF-8">
                    <script src="app.js" defer></script>
                    <title>Tareas</title>
                </head>
                <body>
                    <h2>Tareas</h2>

                    <input type="text" id="tarea" placeholder="Tarea">
                    
                    <select id="tipo">
                        <option value="normal">Normal</option>

                    </select>

                    <button onclick="crear()">Añadir</button>

                    <ul id="agenda"></ul>
                </body>
            </html>
            ```
        === "app.js"

            ```javascript
            function crear() {
                let texto, tipo, li;

                // 1. Obtener el valor (value) del elemento <input type="text" id="tarea" placeholder="Tarea">. Almacenar el valor recuperado en la varibale texto.

                // 2. Obtener el valor seleccionado del elemento <select id="tipo">. Almacenar el texto recuperado en la varibale tipo.
                tipo = document.querySelector("#tipo").value;

                // 3. Crear un elemento de tipo <li> y asignarlo a la variable 'li'

                // 4. Asignar al elemento 'li' creado en el punto 3 el texto obtenido en el punto 1.

                // 5. Si tipo es igual a 'urgente', añadir un color a la variable li (li.style.color = "red";)

                // 6. Si tipo es igual a 'urgente', añadir el elemento <li> creado en el punto 2 al principio del elemento <ul id="agenda"></ul>, sino,
                // el elemento <li> se debe añadir al final del elemento <ul id="agenda"></ul>. Utilizar append y prepend.

                // 7. Borrar el texto escrito en el input con id = "producto"    
            }
            ```

        === "SOLUCIÓN"

            ```javascript
            function crear() {
                let texto, tipo, li;

                // 1. Obtener el valor (value) del elemento <input type="text" id="tarea" placeholder="Tarea">. Almacenar el valor recuperado en la varibale texto.
                texto = document.querySelector("#tarea").value;

                // 2. Obtener el valor seleccionado del elemento <select id="tipo">. Almacenar el texto recuperado en la varibale tipo.
                tipo = document.querySelector("#tipo").value;

                // 3. Crear un elemento de tipo <li> y asignarlo a la variable 'li'
                li = document.createElement("li");

                // 4. Asignar al elemento 'li' creado en el punto 3 el texto obtenido en el punto 1.
                li.textContent = texto;

                // 5. Si tipo es igual a 'urgente', añadir un color a la variable li (li.style.color = "red";)
                if (tipo === "urgente") {
                    li.style.color = "red";
                }

                // 6. Si tipo es igual a 'urgente', añadir el elemento <li> creado en el punto 2 al principio del elemento <ul id="agenda"></ul>, sino,
                // el elemento <li> se debe añadir al final del elemento <ul id="agenda"></ul>. Utilizar append y prepend.
                if (tipo === "urgente") {
                    document.querySelector("#agenda").prepend(li);
                } else {
                    document.querySelector("#agenda").append(li);
                }

                // 7. Borrar el texto escrito en el input con id = "producto"    
                document.querySelector("#tarea").value = "";
            }
            ```

            ```javascript
            function eliminar() {
                let i = 0;
                let precio, items;

                // 1. obtener los elementos hijos del elemento <ul id="lista">, los cuales son de tipo <li> y guardarlos en la variable 'items'.
                items = document.querySelector("#lista").children;

                // 2. Recorrer todos los elementos de la variable items>. Para cada uno de ellos, obtener el precio.

                }
            }
            ```
