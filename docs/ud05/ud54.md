# 4. Asignación de manejadores

## 4.1 Asignando manejadores de eventos desde HTML

??? example "Ejemplo 1 - Manejadores desde HTML"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <head>
            <meta charset="UTF-8">
            <script src="app.js" defer></script>
        </head>
        <html>
            <body>
                <p onmouseover="this.style.background='#FF0000';" onmouseout="this.style.background='#FFFFFF';">
                    HOLA
                </p>
            </body>
        </html>
        ```

## 4.2 Asignando manejadores de eventos desde JavaScript

??? example "Ejemplo 1 - Manejadores desde JavaScript"
                
    === "index.html"

        ```html
        <!DOCTYPE html>
        <head>
            <meta charset="UTF-8">
            <script src="app.js" defer></script>
        </head>
        <html>
            <body>
                <p id="hola">HOLA</p>
            </body>
        </html>
        ```

    === "app.js"

        ```javascript
        document.querySelector('#hola').onmouseover = function() {
            this.style.background = '#FF0000';
        };

        document.querySelector('#hola').onmouseout = function() {
            this.style.background = '#FFFFFF';
        };
        ```
