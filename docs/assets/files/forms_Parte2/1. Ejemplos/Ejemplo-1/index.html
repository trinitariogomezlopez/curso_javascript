⚡ Ejercicio — Eventos de Formulario

Cada elemento dispara un evento que muestra u oculta contenido en la página

📌 Ejercicio 1 — Select con evento onchange

Cuando el usuario elige un idioma en la lista, se dispara el evento onchange. La función muestra el panel del idioma elegido y oculta los demás.
elemento.onchange = function() { ... }
🇪🇸 ¡Hola! Has elegido Español.
El español es hablado por más de 580 millones de personas en el mundo.
🇬🇧 Hello! You chose English.
English is the most widely spoken language in the world.
🇫🇷 Bonjour! Tu as choisi le Français.
Le français est parlé par plus de 300 millions de personnes.

📌 Ejercicio 2 — Radio con evento onchange

Cuando el usuario selecciona una forma de pago, el evento onchange del radio muestra el panel con las instrucciones de esa forma de pago y oculta los demás.
elemento.onchange = function() { ... }
💳 Tarjeta bancaria: Introduce los datos de tu tarjeta.
🏦 Transferencia bancaria: Introduce tu número de cuenta para confirmar el pago.
💵 Efectivo: Pasa por nuestra tienda en horario de 9:00 a 20:00 y muestra el número de pedido.

📌 Ejercicio 3 — Checkbox con evento onchange

Cada checkbox activa o desactiva su panel correspondiente. Si el checkbox está marcado → se muestra el panel. Si está desmarcado → se oculta.
elemento.onchange = function() { if (this.checked) { ... } }
🚀 Envío express activado. Tu pedido llegará en menos de 24 horas.
🎁 Envoltorio de regalo activado. Incluye lazo y tarjeta personalizada.
🛡️ Seguro de devolución activado. Puedes devolver el producto en 30 días sin coste.
