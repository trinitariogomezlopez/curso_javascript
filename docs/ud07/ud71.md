# 1. Efectos con imágenes

Una imagen en HTML se inserta con una sola etiqueta. No tiene etiqueta de cierre. Los atributos más importantes son `src` (ruta de la imagen) y `alt` (texto alternativo para personas con dificultad visual o cuando la imagen no carga).

```html
<!-- La imagen más sencilla del mundo --> 
<img src="foto.jpg" alt="Mi foto de vacaciones"> 

<!-- Con tamaño definido --> 
<img src="foto.jpg" alt="Mi foto" width="300" height="200">
```

            ```javascript
            let tarjetas = document.querySelectorAll('.tarjeta');
            let i = 0;

            /*
                ENUNCIADO: 
                        1) Se deben recorrer todas las tarjetas (variable 'tarjetas'). Para cada tarjeta, añadir un evento que se ejecutará cuando se pulse sobre la tarjeta. 
                        Cuando se ejecute el evento, se añadirá la clase CSS "volteada" a la tarjeta que ha sido pulsado con el ratón. Emplear toggle.
            */
            while(i < tarjetas.length){
                tarjetas[i].addEventListener('click', function () {
                    this.classList.toggle('volteada');
                });

                i++;
            }
            ```
